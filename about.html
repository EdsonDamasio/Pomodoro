O que é o Pomodoro?

    A técnica Pomodro foi desenvolvida em 1988 pelo italiano Francisco Cirillo, é um método de gestão de tempo que pode ser aplicado para diversas tarefas, dentre elas o estudos.

Como funciona

    O método Pomodoro é simples e dura duas horas. Primeiro, você realiza uma atividade durante 25 minutos. Quando acabar o tempo, descansa 5 minutos. Assim sucessivamente até que complete as duas horas. Como recompensa, você descansa mais 30 minutos.

Vantagens e benifícios

  • Alivia a ansiedade
  • Aumenta o foco e a concentração por meio da redução das interrupções
  • Aumenta a conscientização das decisões
  • Aumenta a motivação e mantém-na constante
  • Reforça a determinação para atingir os objetivos
  • Melhora o processo de trabalho ou estudo

    • Porque fiz este projeto?

      O motivo pelo qual fiz este projeto da página pomodoro foi para por em prática minhas habilidades como recém estudante no ramo da programação e no curso ciência da computação em seu primeiro ano de estudo na Universidade Jorge Amado. As principais competências nesta página foram os conceitos de CSS (cascading style sheets) como agregar vídeo animado de fundo, uma interface simples para o usuário navegar e cores para a página ser chamativa e intuitiva. HTML (Hyper Text Mark linguage), para formar toda a estrutura da página agregando a navbar e suas devidas funções como redirecionar para páginas e ação do E o principal deles o Java Script cujo fará a função de fazer o pomodoro funcionar através de manipulações feita no DOM e eventos para ativar a função de contagem do pomodoro. Alem dessas 3 linguagens utilizadas para a construção deste projeto, a responsividade foi incluída para a página seja compatível com os dispositivos móveis (celulares), tornando a navegação e a experiência mais acessível para todos os usuários.
